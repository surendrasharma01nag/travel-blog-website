## Getting Started

### View Locally
Simply open `src/index.html` in your web browser to view the website.

### Live Preview
Visit the GitHub repository: [travel-blog-website](https://github.com/surendrasharma01nag/travel-blog-website)

## Page Sections

1. **Header Navigation**
   - Logo and brand name (Wanderlust)
   - Navigation links (Home, Destinations, Stories, About)
   - Search and menu icons
   - User avatar

2. **Breadcrumb Navigation**
   - Stories / The Enchanted Forest

3. **Article Content**
   - Title: "The Enchanted Forest: A Journey Through the Emerald Canopy"
   - Author byline: Amelia Stone
   - Publication date: July 15, 2023
   - Hero image (640px height)
   - Four paragraphs of rich content

4. **Social Stats**
   - 234 Likes
   - 56 Comments
   - 123 Shares

5. **Comments Section**
   - Three existing comments with avatars
   - Comment form with textarea
   - Post Comment button

## Responsive Breakpoints

- **Desktop**: 1280px+ (full layout)
- **Tablet**: 768px - 1024px (adjusted padding)
- **Mobile**: < 768px (stacked navigation, adjusted image height)
- **Small Mobile**: < 480px (compact layout)

## Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with Flexbox
- **Google Fonts** - Newsreader font family
- **SVG Icons** - Scalable vector graphics
- **PNG Images** - High-quality photos
